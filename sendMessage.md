There is a script called send_message.py that triggers a message to go to admins. You call it like this. If you need to include a newline character, you can use <BR>.

These updates are meant to be for notable events (I set a new record for R2 in a model; I have finished a long-running task; I would not expect to be getting many messages, so use this sparingly; these aren't for logs, they are for **big news**)

python "C:\Users\zcapo\Documents\workspace\send_message.py" -msg "[message goes here]" --telegram-alert
